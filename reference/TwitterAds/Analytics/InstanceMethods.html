Instance Method Details

- (Array) stats(metrics, opts = {})

Pulls a list of metrics for the current object instance.

Examples:

metrics = [:promoted_tweet_timeline_clicks, :promoted_tweet_search_clicks]
object.stats(metrics)

Parameters:

  • metrics (Array) —

    A collection of valid metrics to be fetched.

  • opts (Hash) (defaults to: {}) —

    An optional Hash of extended options.

Options Hash (opts):

  • :start_time (Time) —

    The starting time to use (default: 7 days ago).

  • :end_time (Time) —

    The end time to use (default: now).

  • :granularity (Symbol) —

    The granularity to use (default: :hour).

  • :segmentation_type (Symbol) —

    The segmentation type to use (default: none).

Returns:

  • (Array) —

    The collection of stats requested.
