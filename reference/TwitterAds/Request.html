Parameters:

  • client (Client) —

    The Client object instance.

  • method (Symbol) —

    The HTTP method to be used.

  • resource (String) —

    The resource path for the request.

  • opts (Hash) (defaults to: {}) —

    An optional Hash of extended options.

Options Hash (opts):
